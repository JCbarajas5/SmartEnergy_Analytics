# household_power_consumption
Detección de Anomalías en Consumo Energético + Recomendaciones de Ahorro Problema de decisión ¿Qué edificios/hogares tienen consumo anómalo y cuánto podrían ahorrar? Datasets  UCI: Individual Household Electric Power Consumption (2M+ mediciones, 4 años) Kaggle: Smart Meters London (5,500 hogares, datos horarios)  Por qué es bueno  ✅ Tema actual (sustentabilidad) ✅ Pocos compañeros lo harán ✅ Combina: regresión (predicción de consumo), detección de anomalías, clustering (perfiles de consumo) ✅ Dashboard potente: "Tu consumo vs. hogares similares + potencial de ahorro"  Modelos a comparar  ARIMA/Prophet (series temporales baseline) Random Forest Regressor Isolation Forest (detección de anomalías) K-means para agrupar por patrón de consumo  Métrica de negocio kWh ahorrados × tarifa = impacto económico
